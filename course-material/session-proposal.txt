
20min Microlearning Session
===========================

4 topics a ~5mins

We orient ourselves on very basic concepts of programming.
Target users should have little to no experience in programming
and its related concepts.

Statements on the amount of slides per topic relate to
"full-presentation-shortened" inthe 
/course-material/preprocessed/slides directory.


Topic A (7 slides)
-------------------
Variables, simple data types
o definition of variables
o overview on data types
o integers & strings (basics)


Topic B (5 slides)
------------------
If-statement basics
o definition and examples (code snippets)


Topic C (7 slides)
------------------
Loops
o for loops (basics)
o while loops (basics)
o loop keywords (break, pass, continue)


Topic D (7 slides)
------------------
Functions
o basic concept
o definition
o examples for valid use and errors
